  1. 1. 常用命令：

screen

Ye
2023-08-14

screen 是一个非常有用的命令，提供从单个 SSH 会话中使用多个 shell 窗口（会话）的能力。当会话被分离或网络中断时，screen 会话中启动的进程仍将运行，你可以随时重新连接到 screen 会话。如果你想运行一个持久的进程或者从多个位置连接到 shell 会话，这也很方便。

常用命令：

  • 新建 session
    screen -S <name>

  • detach 子窗口
    Ctrl+a+d

  • 列出当前所有的 session
    screen -ls

  • 再次进入 session
    screen -r <name>

  • 删除 session
    screen -S <name> -X quit
